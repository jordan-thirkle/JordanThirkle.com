Neon Forge is a passion project born out of a love for high-octane roguelites. The goal was to create a fluid, responsive combat system paired with a neon-drenched aesthetic. 

## Overview
I handled all aspects of development, from the core C# architecture to the shader graph visual effects. The game requires precise timing and aggressive playstyles to survive the procedural arenas.

## Key Features
* Custom procedural level generation algorithm
* Highly optimized particle systems for 60fps on low-end hardware
* Integration with Steamworks API for achievements and cloud saves
