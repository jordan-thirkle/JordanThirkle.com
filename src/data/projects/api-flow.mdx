After building dozens of micro-SaaS apps, I got tired of writing the same Express.js boilerplate. API Flow allows users to visually connect database nodes to generate deployable API routes in seconds.

## Overview
The platform leverages React Flow for the node-based canvas. Users define their database schema visually, and the backend engine compiles it into production-ready endpoints.

## Key Features
* Drag-and-drop node interface built with React Flow
* Automated OpenAPI/Swagger documentation generation
* One-click deployment to Vercel/AWS
