# Creator Hub Platform

The **Creator Hub** project was born out of a need for a unified space to showcase both code and content. 

## Key Features
- **CMD+K Search**: Global command palette for navigation.
- **Engagement Dock**: Persistent floating bar for likes and social sharing.
- **Dynamic Content**: Powered by Astro Content Collections.

The architecture focuses on scalability and extreme speed.
